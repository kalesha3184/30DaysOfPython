# Day 2: String Methods & Text Manipulation

## What I Learned

### String Methods:
- `.strip()` - Removes spaces from beginning and end
- `.upper()` / `.lower()` - Changes case
- `.capitalize()` - First letter capital, rest lowercase
- `.replace()` - Swaps text
- `len()` - Counts characters

### String Slicing:
- `[start:end]` - Get specific range
- `[start:]` - From start to end
- `[:end]` - From beginning to end
- `[::-1]` - Reverse string

## My Project: Username Validator

Validates usernames by:
1. Removing extra spaces
2. Converting to lowercase
3. Checking length (5-15 characters)
