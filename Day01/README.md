# Day 1: Variables, Data Types & First Program


## What I Learned
- Variables store data (like boxes with labels)
- 4 basic types: string (text), int (whole numbers), float (decimals), bool (True/False)
- 'print()' shows output, 'input()' gets user input
- f-strings let you embed variables in text:'f"hello {name}"'
